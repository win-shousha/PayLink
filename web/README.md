# Frontend PayLink - Vue 3

Frontend web de la plateforme PayLink construite avec Vue 3 et Vite.

## Installation

```bash
cd web
npm install
```

## Développement

```bash
npm run dev
```

L'application sera disponible sur `http://localhost:5173`

## Build

```bash
npm run build
```

## Structure du projet

- `src/pages/` - Pages Vue
- `src/components/` - Composants réutilisables
- `src/stores/` - Stores Pinia
- `src/router/` - Configuration du routeur
- `src/services/` - Services API
- `src/style.css` - Styles globaux

## Variables d'environnement

Copier `.env.example` vers `.env` et configurer l'URL de l'API.
